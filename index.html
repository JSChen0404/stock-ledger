如何設定雲端（Worker + KV）
一次性設定：
1. 到 Cloudflare 免費註冊 → Workers & Pages → Create Worker。
2. 貼上附帶的 worker.js，把裡面的 SECRET 改成你自己的密碼，Deploy。
3. 左側 KV → 建一個 namespace（例如 ledger）→ 回 Worker 的 Settings → Bindings → 新增 KV binding，變數名 KV、選剛建的 namespace。
4. 複製 Worker 網址與你設的密碼，填到上面的欄位，按「測試連線」。

之後交易資料就存在你的 KV（永久保存、可多裝置），盤中自動抓即時報價。資料只在你的 Cloudflare 帳號裡，密碼別人不知道就存取不了。
